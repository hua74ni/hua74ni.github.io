最开始的思路：
    很简单的做法，只有一个篮子，而且篮子只有存放一个物品，并且同时只能一个人操作。
    依据这个思路，写出来的代码就是不管生产者、还是消费者在操作前都需要获取锁的操作。

优化思路：

    public static void main(String[] args) {

        LinkedBlockingQueue<String> goods = new LinkedBlockingQueue<String>();

        new Thread(new Producer(goods), "A类型").start();
        new Thread(new Producer(goods), "B类型").start();

        new Thread(new Consumer(goods), "消费者a").start();
        new Thread(new Consumer(goods), "消费者b").start();

    }
